Generate Product Catalogue

The Customizable Product Catalogue Maker in Odoo Helps User to Easily Create and Customize Product Catalogues to Fit Their Needs. It Has a Simple Interface and Lots of Options to Customize. This Makes It Easy to Create Nice-Looking Catalogues to Share with Customers.


Features

Only Authorized Users have Access to Generate the Product Catalogue.

Generate Catalogue for Selected Products or Categories.

If no Products or Categories are Selected, then All Products or Categories are Considered for Generating the Catalogue.

User Has Five Different Design Options to Generate Product Catalogue.

User Can Print Images on the Product Catalogue in Three Sizes: Small, Medium, and Large.

User Can Print Product Price in Selected Currency and Pricelist.

User Customize Product Catalogue by Selecting Details to Include: Price, Description, Unit of Measure, Internal Reference, Website Link, Category, and Image.

How Application Work
